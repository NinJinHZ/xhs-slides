# xhs-slides

生成小红书3:4比例PPT风格幻灯片网页的 Claude Code Skill。

将 Markdown 文章内容转换为可交互的 HTML 演示文稿，支持多种视觉风格，并可导出 PNG 图片。

## 功能特点

- 📐 **3:4 比例** - 完美适配小红书封面和笔记
- 🎨 **11 种风格** - 科技蓝、粉紫、橙红、暗黑、白底、金色、氧气、纯净、商务橙、自然绿、电影
- ⚡ **一键生成** - 只需提供 Markdown 内容
- 📤 **PNG 导出** - 自动截取每张幻灯片为图片
- 🎬 **动画效果** - 流畅的入场动画

## 使用方式

```bash
/xhs-slides

# 粘贴 Markdown 文章内容
# 指定风格（可选）
```

## 示例

```
风格：电影

# 我的文章标题

## 第一章
内容...
```

## 风格预览

| 风格 | 特点 | 适用场景 |
|------|------|---------|
| 科技蓝 | 深蓝+青蓝 | 科技、AI、工具评测 |
| 粉紫 | 深紫+粉紫 | 女性向、创意类 |
| 橙红 | 深橙+暖色 | 励志、创业、电商 |
| 暗黑 | 纯黑+白色 | 开发者、极客 |
| 白底 | 白色+黑色 | 知识干货、教程 |
| 金色 | 深棕+金色 | 高端品牌、商业 |
| 氧气 | 白色+粉绿 | 小红书、清新可爱 |
| 纯净 | 浅灰+黑色 | 极简、干货、教程 |
| 商务橙 | 白底+橙色 | 商业、创业、效率 |
| 自然绿 | 白底+绿色 | 清单、健康、生活方式 |
| 电影 | 深色+动画 | 演讲、演示、发布会 |
